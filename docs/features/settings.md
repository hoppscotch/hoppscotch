# Settings

The Settings page allows you to customize your Hoppscotch experience and configure various options for your API requests.

## General Settings

Configure general application behavior, theme, and language.

## Interceptors

Interceptors are used to bypass browser limitations like CORS. You can choose between:
- **Browser**: The default interceptor, subject to browser security policies.
- **Proxy**: Uses a proxy server to relay requests.
- **Extension**: Uses the Hoppscotch Browser Extension for enhanced capabilities.
